# OpenFractionCalc
----------------------------------
OpenFractionCalc is a simple fractional calculator

-----------------

|  Features  |
| ------- |
| Add (+) |
| Subtract (-) |
| Multiply (×) |
| Divide (÷) | 
| Exponent (^) |
| Square Root (√) |
| Fraction to decimal |
| Round |
| Floor |
| Ceiling |
